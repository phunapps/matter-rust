# matter-crypto

Matter session establishment — PASE (commissioning) and CASE (operational).

Part of [`matter-rust`](https://github.com/phunapps/matter-rust). Milestones 3
and 4.

> Status: **pre-release (`0.0.0`)**, not yet implemented.

## Cryptographic review requirement

Any release of this crate that changes the protocol behaviour requires an
external review by a cryptographic engineer **before** publishing to crates.io.
The maintainer enforces this manually. Do not bypass it.
