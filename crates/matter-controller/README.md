# matter-controller

The high-level Matter controller API — the crate consumers will actually
depend on. Wraps every other `matter-*` crate behind a small, async-friendly
surface.

Part of [`matter-rust`](https://github.com/phunapps/matter-rust). Milestone 8
(the v1.0 release).

> Status: **pre-release (`0.0.0`)**, not yet implemented.
